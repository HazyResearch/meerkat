# Reactive Functions in Meerkat

Meerkat provides some reactive functions out of the box.



| Reactive Functions |
| ------------------ |
| *Type Casting* |
| `mk.bool`, `mk.int`, `mk.float`, `mk.str` |
|*Object Creation* |
| `mk.tuple`, `mk.list`, `mk.dict`, `mk.set` |
| *General* |
| `mk.len`, `mk.sum`, `mk.range` |
