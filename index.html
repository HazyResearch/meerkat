Foundation Models meet Data and Apps.

Meerkat is an open-source Python library to wrangle unstructured and structured data using dataframes, connect data to foundation models, and build full scale web apps on top (or just run interfaces in a notebook).

Install Meerkat

pip install meerkat-ml

What can you do with Meerkat?

Dataframes for Unstructured Data

Images, Text, Audio, Video, and more.

Load any unstructured data into a Meerkat dataframe and use a simple API to wrangle data. 


import meerkat as mk
df = mk.DataFrame(...)
df.map(...)

We're building support for running foundation models on your data, right in the dataframe.
